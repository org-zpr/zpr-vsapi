# zpr-vsapi

This repository contains the IDL description of the ZPR Visa Service API.

This is pulled into ZPR projects as a submodule in the zpr-common repository.

> **Pre-Release / Beta Notice**
>
> This repository is in active, early-stage development. The Cap'n Proto schemas contained here are **not stable** and should be treated as beta-quality work.
>
> - **Breaking changes may occur at any time** without prior notice or deprecation periods.
> - Schema definitions, field names, types, and overall structure are subject to significant revision.
> - Do **not** rely on these definitions for production systems until a stable release is announced.
>
> Feedback and contributions are welcome, but please be aware that any work built on top of these schemas may need to be updated as the definitions evolve.
